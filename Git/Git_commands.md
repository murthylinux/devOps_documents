# GIT Commands:
```
* git init
* git clone <repo_url>
* git status
* git add <file>
* git add .
* git commit -m “message”
* git branch
* git branch <<branch>>
* git checkout <<branch>>
* git merge <<branch>>
* git remote -v
* git pull origin <<branch>>
* git push origin <branch>
* git log
* git log –oneline
* git revert <<commit>>
* git reset <<commit>>
* git tag
* git tag -a v1.0 -m “tag”
* git config –global user.name “name”
* git config –global user.email “email”
```
